### 🚀 Мои проекты и репозитории

- **[FinTwist (AI Financial Assistant)](https://github.com/timgank/fin_twist)**
  Умный Telegram-бот для трекинга финансов. Автоматически парсит чеки с помощью EasyOCR и pyzbar, а также извлекает траты из свободного текста, используя локальную LLM Llama 3.2 через Ollama. Поддерживает экспорт аналитики в Excel (pandas) и генерацию графиков (matplotlib).

- **[AI Photo Finder](https://github.com/timgank/ai-photo-finder)**
  ИИ-сервис поиска фотографий для участников фестивалей. Использует алгоритмы распознавания лиц (Facial Recognition), чтобы быстро находить кадры с нужным человеком, парся множество публичных ссылок облачных хранилищ. Бэкенд построен с использованием FastAPI.

- **[Telegram Voice Bot](https://github.com/timgank/tg_voice_bot)**
  Локальный бот для offline-распознавания голосовых сообщений и аудиофайлов. Построен на базе библиотеки Vosk. Включает ролевую модель с админ-панелью и систему контроля доступа (whitelist) для приватного использования.

- **[SVG Generator](https://github.com/timgank/climbing-svg-generator)**
  Скрипт для программной генерации макетов под лазерную гравировку. Формирует плотный, переплетающийся (interlocking) паттерн для кастомных скалолазных зацепок и интерактивных бордов.

- **[T-Education / Contest Training](https://github.com/timgank/t-education)**
  Репозиторий с решениями алгоритмических задач и скриптами для подготовки к контестам (T-Bank / Tinkoff Education). Практика структур данных и оптимизации алгоритмов на Python.
